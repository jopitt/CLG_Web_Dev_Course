Foundations of Web Development

Lesson 3: Lists, Tables and Images

Lists

  • Lists contain groups of related data.
  • Ordered lists are created using ol tag. Each list item is enclosed in li tags.
  • Unordered lists are created using ul tags. Each list item is enclosed in li tags.
  • The CSS style that controls the bullet or number style used by a list is list-style-type.
  • For unordered lists, list-style-types include:
    • disc
    • circle
    • square
  • For ordered lists, list-style-types include:
    1. decimal
    2. lower-latin
    3. upper-latin
    4. lower-roman
    5. upper-roman

Tables

  • Tables are created with table tags.
  • Tables have rows, enclosed in "tr" tags, which contain cells defined between td tags.
  • Table rows can be grouped together into header thead, body tbody and footer tfoot sections.
  • Example table:
    Items Cost
    Keyboard 40.00
    Mouse 15.00
    Total 55.00

Images

  • Images are embedded in a page using an empty img tag. The src and alt attributes set the location of the image file and alternative text to describe the image.
  • We can style images by changing the following attributes:
    • height
    • width
    • border
    • padding
    • opacity
    • border-radius.

Borders

  • Borders can have a border-width, border-style and border-color.
  • Border-width can be measured in units like px, em, or keywords thin, medium or thick.
  • Specifying 2 values for border-width specifies the values for top and bottom, followed by right and left borders, e.g. border-width: 1px 10px;.
  • Specifying 3 values for border-width sets the widths of the top, then left and right, then bottom borders.
  • Specifying 4 values for border-width specifies the widths of top, right, bottom and left borders.
  • Border-style sets the line pattern. It can be:
    • solid
    • dashed
    • dotted
    • double
    • none
    • inset
    • outset
    • groove
    • ridge.
  • Border-color sets the border colour.
  • As with border-width, you can set 1, 2, 3 or 4 values for border-style and border-color to set the values for different sides independently.
  • The border shortcut allows you to set width, style and colour in one go, e.g. border: solid 1px red;.
  • The CSS Box Model shows how the overall width and height an element occupies on a page and consists of the element's height and width, the padding, the border and the margin.
  • Specific order of the CSS Box Model is:
    • element contents
    • padding
    • border
    • margin.
