Lesson 8: Responsive Web Design

Responsive

  • Responsive Web Design is often today's standard approach when designing a website.
  • It allows for fluid layouts which respond to a user's device and browser.
  • The viewport is the visible area of the website when viewing through your browser window or device screen.
  • We use the viewport 'meta' tag to define how a user's viewport should render the website width, zoom level and more.

Media Queries

  • A media query is a CSS feature that allows us to apply CSS styles only when a user browses a wwebsite with certain browser viewport sizes, devices, media, or device orientation.
  • We write a media quary with the @media syntax, e.g.
    @media screen and (min-width: 50em) { .container { display: flex; }

Mobile-First Design

  • Building with a mobile-first responsive web design will make your website usable on multiple devices and offer a good user experience on your page, no matter what context in which your users are visiting.
  • When developing with a mobile-first approach, we should set up our CSS rules for mobile as a foundation and then build on that, using things like media queries.
  • We also consider things such as slower possible connection speeds and the use of fingertips instead of the mouse to navigate.
