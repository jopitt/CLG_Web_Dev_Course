Lesson 1: Introduction (heading 2)

What is HTML (heading 3)

Structure, Content, Words (heading 4)

  • (unordered list) HTML stand for Hypertext Markup Language and is the language webpages are written in.
  • HTML documents are comprised of HTML elements.
  • All HTML pages start with a doctype delcration that tells the browers which version of HTML is being used. The lastest version is HTML5.
  • After the doctype declaration, HTML pages then have an html element.
  • Inside the HTML element is a head element where information about the page goes.
  • After the head element there's a body element where the contents of the page goes.
  • Most HTML elements have opening and closing tags.
  • HTML elements can have attributes that add more information.
  • Some HTML elements do not use a closing tag, just attributes e.g. images.
  • A shortcut for comment style is CMD + /.
  • Click on the link to see HTML elements in action: HTML practice.
  • This page lists all the HTML elements, which are created using tags.

What is CSS

Colours, Styles, Sizes

  • CSS stand for Cascading Style Sheets.
  • CSS controls the presentation of a webpage.
  • CSS uses CSS rules to control the presentation of HTML elements. CSS rules have a selector, properties and values.
  • CSS can be added to a webpage using:
    1. (ordered list) inline styles
    2. internal CSS
    3. external stylesheets (recommended)
  • Inline styles add the style attribute to HTML elements.
  • Internal CSS puts CSS inside a style element placed in the head section of a webpage.
  • External stylesheets use a link element in the head section of a webpage. This links to an external .css file that contains CSS rules.

Lesson 2: Links, Colours and Fonts

Links

  • A link is an element a user can click on to be taken to another webpage or other content.
  • Links are created using the anchor tag "a".
  • The href attribute defines a link's destination href="www.netflix.com"> Visit Netflix.
  • The href can specify a relative path for local content, an absolute path to go to a different webpage, or an element on the same page.
    1. Relative: If we had a page in our file structure named about-us.html, we could link to it with the forward slash, href="/about-us.html"
    2. Anchor: Can have target and title attributes. The target attribute can tell a browser to open a link in the same window or in a new one
      • Opens in a new window, href="https://www.w3.org" target="_blank"
      • Title attribute: Can display a tooltip when the user hovers over a link, href="/tree.jpg" title="Click to see a photo of the tree"
  • Links can be in different states depending on the user interaction with them and can be maneuvered with CSS:
    1. unvisited: blue
    2. visited: purple
    3. hover: cursor changes to a hand
    4. active: red

Colours

  • The RGB colour model is used to display colours on screens using red, green and blue light.
  • Colours can be referred to using:
    1. colour names
    2. RGB values
    3. hex codes; or
    4. HSL values.
  • There are 16 basic colour names that all browsers recognise, including older browsers.
  • There is an extended list of over 100 colour names that most modern browsers recognise.
  • RGB values specify red, green and blue component of a colour with a number from 0-255 e.g. rgb(35,126,245).
  • Hex codes use pairs of hexadecimal symbols to specify the red, green and blue components with values from 00-FF e.g. #237ef5
  • HSL values specify the hue (0-360ยบ), % saturation and % lightness of a colour e.g. hsl(214, 91%, 55%).
  • A colour generator can help with creating the perfect palette.

Fonts

  • Some of the main CSS properties that control font formatting and styling are:
    1. colour
    2. font-weight, font-style, text-decoration
    3. font-family
    4. font-size
  • The color property controls text colour.
  • For basic text formatting, we use:
    1. bold, font-weight : bold
    2. italic, font-style : italic
    3. underline, text-decoration : underline
  • The font-family property can specify a generic family like serif, sans-serif, or monospace, or a specific font.
  • We can use standard installed fonts or we can incorporate custom fonts from a site such as Google Fonts.
  • The font-size property can use absolute units like px or pt, or relative units, like em or %, e.g. h2 { font-family: 'Playfair Display'; font-size: 1.2em; font-weight: bold; color: #ffcbd2; }
