Foundations of Web Development

Lesson 1: Introduction

What is HTML

Structure, Content, Words

  • HTML stands for Hypertext Markup Language and is the language webpages are written in.
  • HTML documents are comprised of HTML elements.
  • All HTML pages start with a doctype delcration that tells the browers which version of HTML is being used. The lastest version is HTML5.
  • After the doctype declaration, HTML pages then have an html element.
  • Inside the HTML element is a head element where information about the page goes.
  • After the head element there's a body element where the contents of the page goes.
  • Most HTML elements have opening and closing tags.
  • HTML elements can have attributes that add more information.
  • Some HTML elements do not use a closing tag, just attributes e.g. images.
  • A shortcut for comment style is CMD + /.
  • Click on the link to see HTML elements in action: HTML practice.
  • This page lists all the HTML elements, which are created using tags.

What is CSS

Colours, Styles, Sizes

  • CSS stand for Cascading Style Sheets.
  • CSS controls the presentation of a webpage.
  • CSS uses CSS rules to control the presentation of HTML elements. CSS rules have a selector, properties and values.
  • CSS can be added to a webpage using:
    1. (ordered list) inline styles
    2. internal CSS
    3. external stylesheets (recommended)
  • Inline styles add the style attribute to HTML elements.
  • Internal CSS puts CSS inside a style element placed in the head section of a webpage.
  • External stylesheets use a link element in the head section of a webpage. This links to an external .css file that contains CSS rules.
