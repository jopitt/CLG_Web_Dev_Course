Lesson 5: CSS Selectors and Positioning

CSS Selectors

  • The CSS descendant selector is used to style any element that's within another element, e.g.
    div p { background-color: yellow; }
  • The child selector pattern styles only elements that are direct children of the first element specified, e.g.
    div > p { background-color: yellow; }
    will only style a paragraph element directly after a div element.

Combining selectors

  • We can use descendent or child selector patterns to be more specific by targeting an element's class or ID, e.g.
    #main > .column {
    background-color: yellow;
    }
  • We can keep adding children or descendants to a selector to keep making it even more specific.
  • The selecting rules apply, no matter how many levels deep or how specific you want to be
    #main .times li span {
    background-color: yellow;
    }

CSS positioning

  • The position of elements on a page can be altered with the CSS position property.
  • The position values are:
    • static
    • relative
    • absolute
    • fixed.
  • Position static means the element is in its usual position within the page. This is the default position for all elements. This is the default element.
